# Phishing Basics

This file contains my beginner notes on phishing awareness and basic email analysis.

## What is Phishing?
Phishing is a social engineering attack where an attacker attempts to trick a user into revealing sensitive information, clicking a malicious link, or downloading a harmful attachment.

## Common Signs of a Phishing Email
- Urgent or threatening language
- Misspelled sender names or domains
- Suspicious links
- Unexpected attachments
- Requests for passwords or sensitive data
- Poor grammar or unusual formatting

## Basic Analysis Steps
1. Review the sender address carefully
2. Hover over links before clicking
3. Look for spelling errors or suspicious wording
4. Check whether the request is unusual
5. Inspect attachments carefully
6. Report suspicious emails to the security team

## Why Phishing Matters
Phishing can lead to:
- Credential theft
- Malware infection
- Financial fraud
- Unauthorized access to company systems
